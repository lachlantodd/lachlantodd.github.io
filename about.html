Australian GPA Calculator:About

Hi, I'm Lachlan Todd. I created this tool as a final project for Harvard's free and online CS50x course. It took just under one week of non-stop coding to create. (including icon and background design)

Every university semester I end up spending some time calculating result requirements, so I finally decided to create a persistent calculator. Instead of using other calculators online, I wanted to create a fully dynamic, responsive and interactive website using Javascript.

Hopefully this saves you as much time as it will save me.
